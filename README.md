# DICOR Carbones y Repuestos — Sitio Web Oficial

Sitio web institucional desarrollado para **DICOR Carbones y Repuestos**, empresa ubicada en Córdoba Capital, Argentina, especializada en la venta de carbones y repuestos para herramientas eléctricas y electrodomésticos.

---

## 🌐 Sitio en Producción

[dicorcarbones.com.ar](https://dicorcarbones.com.ar)

---

## 📋 Descripción

El sitio fue desarrollado con el objetivo de brindar una presencia digital moderna y funcional para el negocio, permitiendo a los clientes explorar el catálogo de productos, conocer la empresa y realizar consultas de forma directa.

### Páginas

- **Inicio** — presentación del negocio, propuesta de valor y carrusel de marcas distribuidoras
- **Nuestra Empresa** — historia, misión y valores de DICOR
- **Productos** — catálogo completo con búsqueda y filtrado por categoría
- **Contacto** — formulario de contacto con mapa de ubicación

---

## 🛠️ Stack Tecnológico

### Frontend
- **React** — biblioteca principal de UI
- **React Router DOM** — navegación entre páginas
- **Tailwind CSS** — estilos y diseño responsive
- **Vite** — bundler y servidor de desarrollo
- **Lucide React** — iconografía
- **React Toastify** — notificaciones y feedback al usuario
- **React Helmet Async** — SEO y meta tags dinámicos por página

### Backend
- **Node.js + Express** — servidor y API REST
- **PostgreSQL (Supabase)** — base de datos para almacenamiento de consultas
- **Nodemailer** — envío de notificaciones por email al recibir consultas
- **CORS** — configuración de acceso entre frontend y backend

### Infraestructura
- **Render** — hosting del backend
- **Supabase** — base de datos PostgreSQL en la nube
- **Cloudinary** — almacenamiento y entrega de imágenes
- **GitHub** — control de versiones

---

## ✨ Funcionalidades Destacadas

- Catálogo de productos con búsqueda en tiempo real por nombre, código o categoría
- Carrusel de marcas con scroll infinito y autoplay
- Formulario de contacto con validación, protección honeypot y notificación por email
- Animaciones de entrada en todas las secciones con `IntersectionObserver`
- Diseño completamente responsive (mobile, tablet, desktop)
- SEO optimizado con meta tags en español por página
- Navegación con indicador de página activa

---

## 🏢 Sobre DICOR

DICOR Carbones y Repuestos es una empresa con sede en Córdoba Capital, Argentina, dedicada a la distribución de repuestos para herramientas eléctricas de las principales marcas del mercado: Bosch, DeWalt, Makita, Milwaukee, Black & Decker, Gamma, Hitachi, Skil, entre otras. Ofrece envíos a todo el país y asesoramiento técnico especializado.
